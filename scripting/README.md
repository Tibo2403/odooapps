# Scripting

Ce dossier est destiné à regrouper les scripts d'automatisation, de maintenance, d'installation et d'exploitation liés au dépôt Odoo Apps.

## Arborescence

```text
scripting/
└── README.md
```

## Scripts disponibles

Aucun script supplémentaire n'a été détecté dans le dossier `scripting/` au moment de la création de cette documentation.

Lors de l'ajout de nouveaux scripts, documenter chaque fichier avec :

- son objectif ;
- le langage utilisé ;
- les prérequis ;
- les variables d'environnement nécessaires ;
- un exemple d'exécution ;
- les précautions éventuelles avant lancement.

## Modèle de documentation pour un script

```markdown
### `nom_du_script.ext`

**Objectif :** décrire brièvement le rôle du script.

**Langage :** Bash, Python, PowerShell, etc.

**Prérequis :** dépendances ou configuration nécessaires.

**Exécution :**

```bash
commande d'exemple
```

**Remarques :** points d'attention, impacts possibles, droits nécessaires.
```
